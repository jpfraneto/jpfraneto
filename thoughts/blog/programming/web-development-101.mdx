# HTML

Acronym for Hyper Text Markdown Language. It is not presentational... It doesn't care how things look. It is a language that cares about what things _mean_. The browser will understand how things look using css. It will understand how things behave using javascript.

Block elements don't allow other elements to sit next to them.
Inline elements allow other elements to sit next to them.
